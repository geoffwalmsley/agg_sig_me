Tic-Tac-Toe

EDIT This is a first go at creating a game of tic-tac-toe on the chia blockchain. When I wrote it the chialisp seemed to be working, but I ran into trouble with the signatures and so my focus has shifted to that.

First thing is to build a list that represents the board. It's just a list of 9 zeros, so row 1 is positions 0 to 2, row 2 is positions 3 to 5 and row 3 is positions 6 to 8.

When a player moves we want to update the list to hold a value representing their move. To replace the $n$th value in a list, we define a recusrive function called setnth. This function will take a list and index and a value and return a new list with the new_value replacing whatever was in the index position of the original list.

setnth will only change the value if it is 0 (indicating no player has moved there yet).

A getnth function is also built for good measure, and will be used in determining winning conditions..

    (defconstant board (0 100 100 200 200 0 0 0 0))

    (defconstant winlines ( (0 1 2) (3 4 5) (6 7 8) ;; rows
                            (0 3 6) (1 4 7) (2 5 8) ;; cols
                            (0 4 8) (2 4 6)         ;; diags
                           ))

    (defun getnth (vals index)
      (if (= index 0)
          (f vals)
          (getnth (r vals) (- index 1))
      )
    )

    (defun setnth (vals index new_val)
      (if (= index 0)
        (if (= (f vals) 0)
          (if (r vals)
            (c new_val (r vals))
            (list new_val)
          )
          (x)
        )
        (c (f vals) (setnth (r vals) (- index 1) new_val))
      )
    )

    (defun check_line (row val board)
      (if (= (getnth board (f row)) val)
        (if (= (getnth board (f (r row))) val)
          (if (= (getnth board (f (r (r row)))) val)
            (q . "winner")
            ()
          )
          ()
        )
        ()
     )
   )

   (defun count_moves (board)
     (if (r board)
         (+ (if (f board) 1 0) (count_moves (r board)))
         (q . 1)
     )
   )

   (defun check_lines (board winlines val)
     (if (check_line (f winlines) val board)
         (q . "winner")
         (if (r winlines)
             (check_lines board (r winlines) val)
             ()
          )
     ) 
   )
