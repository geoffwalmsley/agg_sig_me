Datalayer for MacOS

This is a basic set of steps for getting datalayer up and running on MacOS. I'm running an older macbbook pro, nothing special.

Prerequisites

  • Python 3.8, 3.9, or 3.10
  • Chia Testnet node

Installation

Make sure you're running a recent version of chia, i.e. chia version should be >= 1.6.0. If not, go ahead and update it.

Start a fresh wallet: chia keys generate, and fund it from the testnet faucet.

In order to catch any issues I recommend running a second terminal window to display the chia logs. If there are any problems you'll catch them there. use something like tail -f ~/.chia/logs/debug.log | grep WARN

Starting data layer

In the official docs there is are instructions about port forwarding and firewalls. For the most basic use we don't want to have to fuck about with firewalls, so we're going to ignore any of that stuff, and only open ports if/when needed.

Run the following commands to start datalayer: 

chia start data
chia start data_layer_http

Launch a datalayer with a fee: 

chia data create_data_store -m 0.00000001

This should output a large chunk of JSON data like so: 

{'id': '9ffeca16f4c2727e9304c5973e6570249e6bef0ccf26723f7da902fd8dee3e82', 'success': True, 'txs': [{'additions': [{'amount': 1, 'parent_coin_info': '0x13fcf7de9d670ae0fa2c0bbc6357c9af51b15a580dfc3b437a1ed0ed2faada06', 'puzzle_hash': '0xeff07522495060c066f66f32acc2a77e3a3e737aca8baea4d1a64ea4cdc13da9'}, ..... ]}]}

If this does not run, check the logs for any errors. Otherwise grab the ID and store it somewhere you can easily retrieve it.
